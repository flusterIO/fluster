# Syntax Reference

## Tag

This will generate a tag in your mdx note that can be used to group related thoughts. This is useful both for navigation and for providing valuable information to Fluster's AI.

```mdx
This is my [[#my_tag]] here.
```


## Citation

This will embed a citation in your note formatted according to the optional `.csl` file in your notes directory. 

```mdx
This is my [[cite:my_citation_id]] here.
```
